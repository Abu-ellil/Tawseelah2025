# تطبيق توصيلة للعملاء

هذا هو تطبيق العميل لمنصة توصيلة، وهو يتيح للعملاء تصفح المتاجر، والبحث عن المنتجات، وإنشاء طلبات التوصيل.

## الميزات

- تصفح المتاجر حسب الموقع
- البحث في المنتجات
- سلة الشراء
- تتبع الطلب real-time
- تقييم الخدمة
- محفظة إلكترونية / كل أنظمة الدفع
- إشعارات live

## التقنيات المستخدمة

- React Native
- Expo
- NativeWind (للتخطيط والتصميم)
- Redux Toolkit (لإدارة الحالة)
- Socket.io-client (للوظائف الحية)
- مكتبات أخرى مذكورة في package.json

## الإعداد

1. تثبيت التبعيات:
```bash
npm install
```

2. تشغيل التطبيق:
```bash
npx expo start
```

## بنية الملفات

```
src/
├── components/     # المكونات القابلة لإعادة الاستخدام
├── screens/        # شاشات التطبيق
├── navigation/     # توجيه التطبيق
├── store/          # Redux store وslices
├── services/       # خدمات API وSocket
├── utils/          # وظائف مساعدة وبيانات وهمية
├── assets/         # الأصول مثل الصور والملفات الثابتة
├── theme/          # إعدادات السمة والتصميم
└── types/          # تعريفات TypeScript (إن وجدت)
```

## المكونات الأساسية

- **Redux Store**: يحتوي على_slices التالية:
  - auth: إدارة تسجيل الدخول/الخروج
  - cart: إدارة سلة التسوق
  - orders: إدارة الطلبات
  - stores: إدارة المتاجر
  - products: إدارة المنتجات
  - theme: إدارة السمات (نهاري/ليلي)

- **Theme Context**: يوفر سمة ديناميكية تدعم الوضع النهاري/الليلي واللغة العربية/الإنجليزية

- **Mock Data**: يوفر بيانات تجريبية للتطوير قبل دمج API الفعلي

## التكامل مع الخادم

التطبيق يحتوي على نقاط تكامل جاهزة للاتصال بخادم API:
- ملف `src/services/api.js` يحتوي على وظائف API
- ملف `src/services/socketService.js` يحتوي على وظائف Socket.IO

## ملاحظات
